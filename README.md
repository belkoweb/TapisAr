# Augmented Reality Application with OpenCV

How To Work:

 * install .apk file 
 * Select 'Next Marker' on option menu
 * scan 'starry_night.jpg' and 'akbar_hunting_with_cheetahs.jpg' on drawable with camera
 * see 3D cube over camera
